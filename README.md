# WebdriverIOTasks
 
 Prerequisites:
-NPM
-Git

1. Run npm install
2. Running tests:

To run all available tests:
`npx wdio`

To set the number of workers you can add --maxInstances parameter:
`npx wdio --maxInstances 2`


If you like to run specific test files you can add a --spec parameter:
`npx wdio --spec task1`
